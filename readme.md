## Install

```
$ npm install --global
```


## Usage

```
Usage
	$ code-push-promote -v [version] -k [key] -a [app] -d [deployment]

Options
	-v    App version used in the update check.
	-k    Source deployment key.
	-a    Destination app.
	-d    Destination deployment name.


Examples
	$ code-push-promote -k 12hM3840Fes18273c407zz612Su87U36498my -a MyAppName -d Production
```
